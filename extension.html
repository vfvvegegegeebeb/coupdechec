CoupDEchec — Extension Chrome

Analysez vos parties chess.com en temps réel avec Stockfish, adapté à votre niveau ELO. Aucune donnée ne quitte votre machine.

Télécharger l'extension (.zip)

Version 1.2 · Chrome 88+ · ~1 MB · Stockfish local

Fonctionnalités

Tout ce dont vous avez besoin

⚡

Analyse en temps réel

Stockfish analyse chaque position automatiquement pendant votre partie. La flèche du meilleur coup s'affiche en quelques millisecondes, sans intervention de votre part.

🎯

Adapté à votre ELO

L'extension lit votre ELO directement sur chess.com et adapte la profondeur d'analyse. Débutant à 800 ELO ou maître à 2000, Stockfish joue à votre niveau pour des coups que vous pouvez comprendre et rejouer.

🏹

Flèche visuelle sur l'échiquier

Le meilleur coup s'affiche directement sur l'échiquier chess.com sous forme de flèche dorée. Pas besoin de lire une notation — vous voyez le coup à jouer immédiatement.

🆓

100% gratuit & local

Aucune inscription, aucun abonnement, aucune donnée envoyée. Stockfish tourne entièrement dans votre navigateur. Pas de serveur, pas de cloud — votre vie privée est protégée.

Installation

4 étapes, 2 minutes

1
📦

Télécharger le dossier

Cliquez sur le bouton de téléchargement ci-dessus pour obtenir le fichier .zip. Enregistrez-le, puis extrayez le dossier (clic droit → Extraire tout). Conservez ce dossier dans un endroit fixe sur votre machine.

Le code source est également disponible sur GitHub ↗.

2
🔧

Ouvrir chrome://extensions

Dans Chrome (ou Edge, Brave, Opera), tapez chrome://extensions dans la barre d'adresse et appuyez sur Entrée.

3
🛠️

Activer le Mode développeur

En haut à droite de la page Extensions, activez l'interrupteur "Mode développeur". Trois nouveaux boutons apparaissent en haut à gauche.

4
📂

Charger l'extension

Cliquez sur "Charger l'extension non empaquetée" et sélectionnez le dossier extrait à l'étape 1. L'extension s'installe immédiatement.

Sur chess.com, activez l'analyse via Ctrl+Shift+X — le bouton ♟ Analyse apparaît en bas à droite.

Fonctionne sur Chrome, Edge, Brave et Opera (Chromium 88+). Sur Android : utilisez Kiwi Browser qui supporte les extensions Chrome. Uniquement sur chess.com.
L'extension n'a pas d'icône dans la barre Chrome — c'est intentionnel. Elle se contrôle via le raccourci Ctrl+Shift+X sur une page chess.com. Stockfish tourne entièrement en local — aucune donnée envoyée.
